# Demo App

这是一个演示应用，展示了如何使用 `@yd-hooks/core`、`@yd-libs/core` 和 `@yd-ui/core` 三个包。

## 功能

- ✅ 使用 `useImmer` hook 进行状态管理
- ✅ 使用 `@yd-libs` 工具函数（格式化、验证、存储）
- ✅ 使用 `@yd-ui` UI 组件（Button、Input、Card）
- ✅ 本地存储功能演示

## 运行

```bash
# 在根目录安装依赖
pnpm install

# 启动开发服务器
pnpm --filter demo-app dev

# 或者直接进入目录
cd apps/demo-app
pnpm dev
```

## 技术栈

- React 18
- TypeScript
- Vite
- @yd-hooks/core
- @yd-libs/core
- @yd-ui/core
